고지서 이미지(JPEG, PNG)를 업로드하면 자동으로 정보를 추출합니다.

API 키 발급 방법:
1. 네이버 클라우드 플랫폼 접속 → 회원가입/로그인
2. 콘솔 → AI·NAVER API → AI Service → CLOVA OCR → 이용 신청
3. General OCR 도메인 생성 (Template OCR 아님!)
4. 생성된 도메인 클릭 → "APIGW Invoke URL" 전체 복사
(예: https://xxxxx.apigw.ntruss.com/custom/v1/xxxxx/xxxxxxxx)
5. 같은 페이지에서 "Secret Key" 복사
6. 위 입력란에 붙여넣고 "API 키 검증" 버튼으로 확인
※ 월 1,000건까지 무료, 이후 건당 약 3원
⚠️ 중요: URL 전체를 복사하세요 (https://부터 끝까지)
