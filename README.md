# 에너지 관리 시스템

웹 기반 에너지 사용량 관리 프로그램

## 설치 방법

1. 의존성 패키지 설치:
```bash
npm install
```

2. 서버 실행:
```bash
npm start
```

3. 개발 모드 실행 (자동 재시작):
```bash
npm run dev
```

## 접속 방법

웹 브라우저에서 `http://localhost:3000` 접속

## 로그인 정보

엑셀 파일(`id, pw.xlsx`)에 저장된 계정 정보로 로그인

예시:
- 아이디: admin
- 비밀번호: 1234

## 기능

- 로그인/로그아웃
- 관리자 대시보드
- 사용자 정보 표시
